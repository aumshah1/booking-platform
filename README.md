# SkyTalk AI

SkyTalk AI is a modern airline booking platform powered by AI.

## Architecture

- **Frontend**: Next.js 15, React, Tailwind CSS, shadcn/ui, Framer Motion
- **Backend**: Node.js, Express, TypeScript
- **Database**: Supabase PostgreSQL

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Backend Setup

1. `cd backend`
2. `npm install`
3. Copy `.env.example` to `.env` and fill in Supabase credentials.
4. `npm run dev`

### Frontend Setup

1. `cd frontend`
2. `npm install`
3. Copy `.env.example` (or `.env.local`) and configure environment variables.
4. `npm run dev`
